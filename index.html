The RAPv1 has in total 41,585 pedestrian samples, each of which is annotated with 72 attributes.

RAPv2

A Richly Annotated Pedestrian Dataset for Person Retrieval in Real Surveillance Scenarios, TIP 2018

Abstract

RAP dataset v2.0, an extended version of RAP dataset v1.0, has 84,928 pedestrian images with 72 annotated attribute and 26,638 of them are also identity annotated. The dataset can be used for pedestrian attribute recognition, person retrieval and person re-identification
